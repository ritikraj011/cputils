## 📂 Projects

### 🔹 [cputils](https://github.com/ritikraj011/cputils)
> A lightweight, single-header **C++ utility library** for Competitive Programming and DSA.

- **Tech Stack:** C++17, CMake, vcpkg
- Zero dependencies — just `#include` and go
- Features: fast I/O, debug macros, GCD/LCM, common typedefs, vector/pair printers
- Installable via `vcpkg` overlay port and `CMake FetchContent`
- Supports Linux, macOS, and Windows

```cpp
#include "cputils.h"

int main() {
    fastIO();
    vector<int> v = {1, 2, 3, 4};
    print1d(v);       // 1 2 3 4
    dbg(v.size());    // v.size() = 4
    ll a = 12, b = 18;
    cout << gcd(a, b); // 6
}
```

> 📦 **Install:** `curl -O https://raw.githubusercontent.com/ritikraj011/cputils/main/include/cputils.h`
